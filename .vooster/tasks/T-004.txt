# 검색 입력 및 결과 리스트 웹 UI(Next.js) 구현

**Task ID:** T-004
**Status:** IN_PROGRESS
**Importance:** MUST
**Complexity:** 4/10
**Urgency:** 9/10
**Dependencies:** T-003

## Description

**설명**: 사용자가 연구 분야를 입력할 수 있는 검색창과, 검색 결과로 논문 리스트(제목, 저자, 간단한 설명)를 카드 또는 리스트 형태로 보여주는 Next.js 기반 페이지를 구현합니다. FastAPI 검색 API와 연동하여 실시간으로 결과를 받아옵니다. UI는 심플하고 반응형으로 설계합니다.
**구현 세부사항**: 1) Next.js 프로젝트 내 검색 입력 컴포넌트 구현, 2) 검색어 입력 시 REST API 호출(fetch/axios), 3) 논문 리스트 결과를 카드/리스트 형태로 렌더링, 4) 로딩/에러 상태 처리, 5) 모바일 및 데스크톱 반응형 스타일 적용, 6) API URL 및 응답 스키마 환경변수로 관리
**테스트 전략**: 1) 다양한 입력에 대해 검색 결과 정상 표시 확인, 2) API 실패/빈 결과/로딩 상태 UI 검증, 3) 크로스 브라우저 및 모바일 테스트, 4) 접근성(ARIA) 체크, 5) E2E 테스트(cypress 등) 작성
